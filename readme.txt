== Description ==

This Widget add Image/banner to the Sidebar. The plugin is used as widget that can be placed in a sidebar of the layout.

== Installation ==

1. Upload `alewb-image-widget.php` to the `/wp-content/plugins/` directory
2. Upload 'css' directory to the '/wp-content/themes/xxxx/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget to the sidebar
5. Customize the widget in the sidebar
